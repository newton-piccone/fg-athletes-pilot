# Full Genomes — Athlete Pilot (Landing Page)

Early-access landing page for the Full Genomes nutrigenomics pilot for athletes.

- **`index.html`** — the entire site (single static page, no build step)
- **`screenshots/`** — real product screenshots used in the phone mockups
- **Live:** https://newton-piccone.github.io/fg-athletes-pilot/

## Activating the early-access form

The form posts to [Formspree](https://formspree.io). Replace `YOUR_FORM_ID` in the
`<form action="...">` tag in `index.html` with your real Formspree form ID, or point
the `action` at a `mailto:` address.
